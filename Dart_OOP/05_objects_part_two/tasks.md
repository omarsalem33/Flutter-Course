## Task 2: Object Reference Experiment
Write a program that demonstrates the difference between reference copy and new object:

1. Create a `Player` class with: name, score, level
2. Create `player1` with some values
3. Assign `player2 = player1` (shared reference)
4. Create `player3 = Player()` (new object) with same values as player1
5. Modify player2's score
6. Print all three players to show that player1 was affected but player3 was not
