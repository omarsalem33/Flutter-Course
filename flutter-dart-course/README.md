# 🐦 Flutter & Dart — Complete Beginner Course

Welcome to the **Flutter & Dart Learning Repository**! This repo is designed for students who are starting from scratch and want to build a strong foundation in Dart programming before diving into Flutter development.

---

## 📚 Course Structure

Each lesson folder contains three files:

| File | Description |
|------|-------------|
| `README.md` | Concept explanation with examples |
| `exercises.dart` | Guided exercises with hints |
| `tasks.dart` | Independent tasks to test your skills |

## 🚀 Getting Started

1. Install the [Dart SDK](https://dart.dev/get-dart)
2. Install [VS Code](https://code.visualstudio.com/)
3. Add the **Dart** and **Flutter** extensions in VS Code
4. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/flutter-dart-course.git
   ```
5. Open any lesson folder and start with `README.md`

---

## 🎯 How to Use This Repo

1. **Read** the lesson `README.md` carefully
2. **Run** the code examples in the explanation
3. **Complete** the `exercises.dart` file (hints are provided)
4. **Solve** the `tasks.dart` independently
5. Move to the next lesson ✅
