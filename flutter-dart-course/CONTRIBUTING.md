# Contributing to Flutter & Dart Course

Thank you for your interest in contributing! Here's how you can help:

## How to Contribute

### 📝 Improve Explanations
- Fix typos or unclear wording in README files
- Add more examples to lessons
- Improve exercise hints

### 🐛 Report Mistakes
- Open an issue describing the error
- Include the lesson number and file

### ➕ Add Content
- New exercises for existing lessons
- Additional tasks with real-world context
- Code solutions (in a separate `solutions/` branch)

## Guidelines

1. Keep explanations **student-friendly** (target: 16-year-olds)
2. All Dart code must be **runnable** (`dart file.dart`)
3. Use **comments** liberally in code
4. Follow the existing structure (README + exercises + tasks)
5. Test your code before submitting

## Folder Structure

Each lesson folder must contain:
```
XX_lesson_name/
├── README.md       ← Explanation
├── exercises.dart  ← Guided exercises with hints
└── tasks.dart      ← Independent tasks
```
