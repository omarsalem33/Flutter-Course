# 01 — Setup Environment

## 🎯 Learning Objectives
By the end of this lesson, you will be able to:
- Understand what tools you need to start developing with Dart and Flutter
- Set up your development environment step by step

---

## 📖 Explanation

Before writing any Dart code, you need to set up your **development environment**. This means installing the right software tools on your computer.

### What You Need

| Tool | Purpose |
|------|---------|
| **Dart SDK** | The core language runtime and tools |
| **VS Code** | Code editor (recommended) |
| **Dart Extension** | Adds Dart support inside VS Code |
| **Flutter SDK** (later) | For building mobile apps |

### Operating System Support

Dart and Flutter work on:
- ✅ **Windows** 10/11
- ✅ **macOS** 10.14+
- ✅ **Linux** (Ubuntu/Debian recommended)

---

## 🔧 Step-by-Step Setup

### Step 1 — Check System Requirements
Make sure your computer meets the minimum requirements:
- At least **4 GB RAM** (8 GB recommended)
- **2 GB free disk space**
- Internet connection

### Step 2 — Install VS Code
Download VS Code from: https://code.visualstudio.com/

### Step 3 — Install Dart SDK
Visit: https://dart.dev/get-dart
Follow the instructions for your operating system.

### Step 4 — Verify Installation
Open a terminal and type:
```bash
dart --version
```
You should see something like:
```
Dart SDK version: 3.x.x
```

---

## 💡 Tips
- Keep your tools **updated** regularly
- Use the **official documentation** when you get stuck
- The terminal/command prompt is your friend!
